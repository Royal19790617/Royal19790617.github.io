java解决maven证书失效问题

问题描述：新建springboot项目，通过maven下载pom.xml引入依赖包时发生报错

  • 报错关键字

解决方案

1.通过chrome浏览器下载证书到本地

  • 在Chrome中打开链接，点击左边“锁”图标后出现弹窗，再点击“证书”。

  • 点击“详细信息”tab，点击“复制到文件”，选择本机某个位置，点击“保存”。
    此处，我保存在d:\ali_maven.cer

2、通过证书添加到java信任证书库

  • 本机管理员打开cmd，切换到：{java安装根目录}\jre\lib\security

  • 输入默认密码：changeit，这是java中cacerts证书库的默认密码。

  • 查看证书，此处注意别名要与导入时一致，密码同上

  • 重启IDE，如果不行需要重启电脑
